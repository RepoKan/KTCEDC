# KTCEDC

Android POS/EDC project for payment terminal development.

## Project Goal

This repository is used for Android POS/EDC application development, including payment flow, device integration, configuration management, and APK build validation.

## Target Environment

- Android
- Kotlin / Java
- POS / EDC terminal devices
- SUNMI device support
- Payment SDK integration

## Development Workflow

All changes must be developed in a separate branch and merged into `main` through Pull Request.

Recommended branch format:

```text
feature/*
fix/*
bootstrap/*
hotfix/*
